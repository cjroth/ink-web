# Basic Example

This is a minimal example showing how to use `ink-web` following the setup instructions from the main README.

## Setup

Install dependencies:

```bash
bun install
```

## Run

Start the development server:

```bash
bun run dev
```

Then open http://localhost:5173 in your browser.

## What's included

- `vite.config.ts` - Vite configuration with all required shim aliases
- `src/main.tsx` - Simple Ink app using the imperative API
- `index.html` - HTML entry point

This example follows the exact setup instructions from the main README.
