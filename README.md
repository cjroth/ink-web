# Thoughtful Starter App

This is a starter repo for creating new cross-platform projects. It comes pre-configured with the following:
- tailwind - ui framework
- shadcn - ui components
- tauri - cross platform desktop + mobile
- vite - bundler - may replace with bun in the future
- react-router (framework mode) - routing / ssg
- typescript - types
- biome - linting / formatting
- fumadocs - docs
